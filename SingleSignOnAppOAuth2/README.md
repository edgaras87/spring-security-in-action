# Notes

### SSO (single sign-on).

## OAuth 2 grants 
 -Authorization code
 -Password
 -Client credentials
 -Refresh token


### Authorization code grants
1. Make the authentication request
2. Obtain an access token
3. Call the protected resource
 
### Password grants
1. Request an access token.
2. Use the access token to call resources.

### Client credentials
1. Request an access token
2. Use the access token to call resources
